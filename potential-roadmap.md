# Potential Roadmap

## Suggested Features

- Per-player bribe cooldowns to prevent players from stacking discounts too quickly.
- Optional permissions for who can leash villagers, open villager menus, or use bribes.
- Configurable sounds and particles for accepted and rejected bribes.
- Admin command to inspect or reset a villager's reputation data.
- Optional storage for bribe history so server staff can audit high-value trades.
